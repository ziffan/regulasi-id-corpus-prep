# Security Policy

## Supported Versions

Only the latest minor version receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2.0 | :x:                |

## Reporting a Vulnerability

DO NOT open a public GitHub issue for security vulnerabilities.

If you discover a security vulnerability, please report it via email to **kampusmerah.com@gmail.com**.

Please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.
- Suggested fix (if any).

We will acknowledge your report within 48 hours and work with you to resolve the issue. We follow coordinated disclosure and ask for 90 days from the report before public disclosure.
